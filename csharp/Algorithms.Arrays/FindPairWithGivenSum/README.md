## Find pair with given sum in an array
 
Given an unsorted array of integers, find one pair with given sum in it.

For example,

Input:
 
arr = [8, 7, 2, 5, 3, 1]

sum = 10


Output:

{i: 0, j: 2} // indexes 0 and 2 -> (8 + 2) 

or

{i: 1, j: 4} // indexes 1 and 4 -> (7 + 3)
